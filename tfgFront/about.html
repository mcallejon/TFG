Acerca de este proyecto





                                                  Este trabajo forma parte del proyecto de fin de grado en la
                                                  especialidad Ingeniería informática. 

                                                  Desarrollado en la Universidad Internacional de la Rioja por 
                                                  M.Callejón, entre los años 2014-2018.
                                              
                                                  El siguiente TFG pretende desarrollar una solución,
                                                  para la monitorización de parámetros ambientales, 
                                                  mediante un dispositivo IoT de bajo coste, basado en Arduino. 
                                                  Con el análisis de los datos obtenidos, se consigue mejorar la 
                                                  prevención de heladas en cultivos productivos, haciendo uso de 
                                                  sistemas de riego por aspersión.
                                              
                                                  El proyecto consta de una placa de desarrollo Arduino con sensores, 
                                                  que serán los encargados de captar los datos. Contará también, 
                                                  con entorno Web para la monitorización en tiempo real de los 
                                                  parámetros, que bajo unas condiciones determinadas, enviará un 
                                                  mensaje SMS de alerta y accionará un sistema de riego, de 
                                                  forma remota.

                                                  El aplicativo web se desarrollará utilizando varias tecnologías 
                                                  como: PHP, MySQL para el modelo, una librería JavaScript para 
                                                  la visualización de datos y BootStrap, CSS y HTML5 para el front-end. 

                                                  El software de control del hardware, estará desarrollado en el IDE 
                                                  de Arduino, que cuenta con un lenguaje de programación cercano a C++.
